t1yOS SDKs

t1yOS 提供齐全的 SDK 帮助用户更便捷的无需编写 API 接口即可操作云数据库以及调用云函数。

JavaScript/TypeScript/小程序/快应用

使用该 SDK 可以在 H5（Web）、HTML、Vue、React、Node.js、React Native、微信小程序、QQ 小程序、头条小程序、抖音小程序、支付宝小程序、快应用等场景下使用，一套 SDK 即可全部搞定！详细使用说明请阅读文档：t1y-sdk-js

Java/Kotlin/Adnroid

使用该 SDK 可以在 Java、Kotlin、Android 等场景下使用，详细使用说明请阅读文档：t1y-sdk-android

Swift/iOS/macOS/watchOS/tvOS

使用该 SDK 支持 Apple 全平台，例如：iOS、macOS、watchOS、tvOS 等，详细使用说明请阅读文档：t1y-sdk-swift

Dart/Flutter

使用该 SDK 可以在 Dart、Flutter 场景下使用，详细使用说明请阅读文档：t1y-sdk-flutter

C#

使用该 SDK 可以在 C# 开发场景中使用，例如：WinForms、WPF、WinUI、MAUI、Avalonia 等 Windows 桌面开发场景以及 Unity、Godot 等游戏开发场景，详细使用说明请阅读文档：t1y-sdk-csharp

Go

使用该 SDK 可以在 Go 语言中进行使用，例如：命令行工具、Wails 跨端开发等等，详细使用说明请阅读文档：t1y-sdk-go

以上 SDK 已经覆盖几乎所有的开发场景，如果你有什么未满足的需求，欢迎联系我们进行反馈，开发人员会第一时间进行开发的～
