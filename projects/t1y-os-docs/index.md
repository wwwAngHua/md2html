# t1yOS 是什么？

t1yOS 是一个 `Serverless` 云操作系统，集 AI、文件系统、云函数、云数据库、定时任务于一体。告别传统云服务器，使用 `JavaScript` 编写云函数，像使用操作系统一样管理云端资源。让开发者专注于创造，而非基础设施。

# t1yOS 有什么？

- 云函数
- 云数据库
- 文件系统
- 网站托管
- 定时任务
- 环境变量
- Meta 元数据
- WebIDE，在线编写代码
- 完整的云操作系统能力
- 企业级端到端全动态加解密
- 原生 AI 能力支持，直接让 AI 接管平台 90+ 能力

用户可直接使用平台 `RESTful API` 或 `SDK` 操作云数据库或调用云函数实现各类功能；同时也支持定时任务定时触发云函数完成复杂任务；除此之外云函数还可以完成绝大部分服务端功能的开发，例如：`JWT` 认证、文件操作、数据库操作、网络请求、支付、`MCP Server` 开发等。

> 值得一提的是，t1yOS 云函数采用纯 `Go` 语言完全自研开发，底层基于 `Go` 语言中最快的 `fasthttp` 框架，并提供类似 `express.js` 框架的设计，内存级云函数，无启动时间，如果你是 `node` 开发者，那么一定会喜欢上 t1yOS 云函数。

# 云函数

t1yOS 云函数采用纯 `Go` 语言完全自研开发，底层基于 `Go` 语言中最快的 `fasthttp` 框架，并提供类似 `express.js` 框架的设计，内存级云函数，无启动时间。具有毫秒级发布、原生支持高并发、性能优异等特点。

t1yOS 云函数具有以下能力：

- 处理 HTTP 请求
- 操作数据库
- 操作文件系统
- 调用第三方服务
- 发送网络请求
- 定时任务等

如需参阅云函数相关文档，请前往：[快速开始](./cloud-function/quick-start/index.html)

# 云数据库

t1yOS 云数据库采用 `NoSQL` 数据库，并且平台自研了一套 `SQL` 转 `BSON` 的系统，不仅仅可以使用原生的 `BSON` 操作云数据库，还能使用 `SQL` 替代一些低频快速的使用场景，非常的方便；并且平台有完整的`索引支持`、支持`云函数`操作云数据库、`RESTfult API` 操作云数据库。同时支持`权限管理`，更精细化的管理各数据库表。

t1yOS 云数据库具有以下能力：

- 完整的云操作系统可视化管理客户端 App
- 云函数操作云数据库
- 精细化的权限管理
- 数据库自动映射为 RESTfult API 供客户端操作
- 完整的索引支持
- 无需编写接口即可直接对云数据库进行增删改查操作

如需参阅云数据库相关文档，请前往：[云数据库文档](./database/index.html)

# MCP Server

使用 `t1yOS MCP Server` 可以让 `AI` 大模型接管 `t1yOS` 平台 `90+` 功能，彻底解放双手，如：帮助你管理云数据库、管理文件系统文件、管理定时任务、管理环境变量等等。

> 例如：帮我查询 `users` 集合中 `name` 字段为王华的数据、帮我检查应用健康状态、帮我看看 `/.private/a.txt` 中都有什么内容、帮我在 `logs` 集合中新建一个 1 天过期自动删除的索引、帮我...

如需参阅 `MCP Server` 相关文档，请前往：[MCP Server](./mcp/index.html)
